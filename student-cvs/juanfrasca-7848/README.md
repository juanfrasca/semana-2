|Command   |Description                        |
|----------|-----------------------------------|
|`pwd`     |Print current working directory    |
|`ls`      |List directory contents            |
|`cd`      |Change current directory           |
|`mkdir`   |Make a new directory               |
|`cp/cp -r`|Copy a file/folder (adding -r)     |
|`rm/rm -r`|Remove a file/folder (adding -r)   |
